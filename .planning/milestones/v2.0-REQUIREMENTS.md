# Requirements: WorkshopPilot.ai

**Defined:** 2026-02-28
**Core Value:** Anyone with a vague idea can produce validated, AI-ready product specs without design thinking knowledge — the AI facilitator replaces the human facilitator.

## v2.0 Requirements

Requirements for v2.0 Dot Voting & Mobile Gate. Each maps to roadmap phases.

### Dot Voting

- [x] **VOTE-01**: Facilitator can configure vote budget per participant (default 2, range 1-8)
- [x] **VOTE-02**: User can place multiple votes on the same Crazy 8s sketch (stacking)
- [x] **VOTE-03**: User can see visual dot indicators on each sketch showing vote count
- [x] **VOTE-04**: User can see remaining vote budget HUD during voting
- [x] **VOTE-05**: Facilitator can open and close voting session
- [x] **VOTE-06**: Votes are anonymous (hidden from others) until facilitator closes voting
- [x] **VOTE-07**: Ranked results revealed simultaneously to all when voting closes
- [x] **VOTE-08**: Facilitator manually selects which ideas advance to Step 9
- [x] **VOTE-09**: User can dot-vote in solo workshops (self-prioritization)
- [x] **VOTE-10**: User can dot-vote in multiplayer workshops with real-time sync
- [x] **VOTE-11**: Facilitator can set countdown timer that auto-closes voting on expiry
- [x] **VOTE-12**: Facilitator can see which participants have placed all votes (completion indicator)
- [x] **VOTE-13**: User can undo a vote by clicking a voted sketch (retract, return to budget)

### Mobile Gate

- [x] **MOBI-01**: Phone users (<768px + coarse pointer) see a dismissible "best on desktop" overlay
- [x] **MOBI-02**: Overlay dismissal persists for the browser session (sessionStorage)
- [x] **MOBI-03**: Gate applies only to workshop pages (not landing, dashboard, pricing)
- [x] **MOBI-04**: Email-to-self CTA with mailto: link containing current URL

## Out of Scope

| Feature | Reason |
|---------|--------|
| Auto-advancement of ideas based on vote count | Removes facilitator judgment; votes inform, do not decide |
| Real-time vote counts visible during open session | Groupthink anchoring trap; NNGroup research is unambiguous |
| Custom vote colors per participant | Defeats anonymous voting by making votes identifiable |
| Hard mobile block with no dismiss | Removes user autonomy; tablet users at 767px would be unfairly blocked |
