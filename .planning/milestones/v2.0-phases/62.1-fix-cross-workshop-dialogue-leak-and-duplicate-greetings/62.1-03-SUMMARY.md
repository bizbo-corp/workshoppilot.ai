## Decisions Made

- **Hard stop vs. soft recovery:** The plan explicitly required a hard stop. The ABSENCE PROTOCOL block is ordered BEFORE the DYNAMIC GREETING in interactionLogic and uses "SUPERSEDES" language to ensure precedence.
- **Sentinel vs. empty string:** An empty string is indistinguishable from "no context loaded yet." The visually-loud `⚠️ NO PRIOR STEP CONTEXT AVAILABLE` sentinel is unmistakable to both the model and developers reading logs.
- **depsRequireContext check:** `Array.isArray(deps) ? deps.length > 0 : deps === 'all'` correctly handles both the array and `'all'` string union types in `STEP_SUMMARY_DEPS`.
