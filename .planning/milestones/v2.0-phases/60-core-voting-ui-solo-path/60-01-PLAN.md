**1. Create `src/components/workshop/voting-hud.tsx`:**

A floating pill HUD component that reads from `useCanvasStore`. The HUD has three states based on `votingSession.status`:

**When status === 'idle':**
- Render a "Start Voting" button pill at `absolute top-3 left-1/2 -translate-x-1/2 z-30`.
- Use shadcn `Button` with variant `default`, size `sm`. Icon: a circle dot or vote icon from Lucide.
- On click: call `openVoting()` from the store.

**When status === 'open':**
- Render the budget HUD pill at same position.
- Show dot glyphs: filled dots `●` for votes placed, empty dots `○` for remaining. Use `Array.from({ length: totalBudget })` mapped to `●` or `○`.
- Show text: `"{remainingBudget} of {totalBudget} remaining"` when `remainingBudget > 0`.
- When `remainingBudget === 0`: show `"All votes placed"` text and a "Close Voting" `Button` (variant `default`, size `sm`).
- Close voting handler: compute results from `dotVotes` + `crazy8sSlots`, then call `closeVoting()` followed by `setVotingResults(results)`. Use the Pattern 3 from 60-RESEARCH.md for result computation:
  - Build a Map of slotId -> voteCount from `dotVotes`
  - Get all slot IDs from `crazy8sSlots.map(s => s.slotId)`
  - Sort by vote count descending
  - Assign ranks with tie handling (same count = same rank)
  - All 8 slots must appear, including zero-vote slots
- Pill styling: `rounded-full bg-background/90 backdrop-blur-sm border shadow-sm px-4 py-2`.

**When status === 'closed':**
- Render nothing (VotingResultsPanel from Plan 02 takes over).

**VoterId derivation:** Use `useUser()` from `@clerk/nextjs`. Set `const voterId = user?.id ?? 'solo-anon'`. Derive `myVotes = dotVotes.filter(v => v.voterId === voterId)` and `remainingBudget = votingSession.voteBudget - myVotes.length`.

**2. Update `src/components/workshop/crazy-8s-grid.tsx`:**

Add voting mode props to `Crazy8sGridProps`:
```typescript
// New optional props for voting mode
votingMode?: boolean;
dotVotes?: DotVote[];
voterId?: string;
remainingBudget?: number;
onCastVote?: (slotId: string) => void;
onRetractVote?: (slotId: string) => void;
```

Import `DotVote` type from `@/lib/canvas/voting-types`.

Inside the slot rendering loop, when `votingMode` is truthy AND `votingSession status` is implied open (parent controls this):

- Compute `slotVoteCount = (dotVotes || []).filter(v => v.slotId === slot.slotId).length` for the badge.
- Compute `myVotesOnSlot = (dotVotes || []).filter(v => v.slotId === slot.slotId && v.voterId === voterId)` for retraction.

Add two elements inside each slot card `div`, AFTER the slot number badge:

**Vote button (bottom-left, "+1"):**
```jsx
{votingMode && (
  <button
    className="absolute bottom-2 left-2 z-10 flex items-center gap-1 rounded-full bg-primary text-primary-foreground px-2.5 py-1 text-xs font-semibold shadow-sm hover:bg-primary/90 transition-colors disabled:opacity-40 disabled:cursor-not-allowed"
    onClick={(e) => { e.stopPropagation(); onCastVote?.(slot.slotId); }}
    disabled={(remainingBudget ?? 0) <= 0}
    title="Cast a vote"
  >
    +1
  </button>
)}
```

**Vote count badge (top-right, clickable for retract):**
```jsx
{votingMode && slotVoteCount > 0 && (
  <button
    className="absolute top-2 right-2 z-10 w-7 h-7 rounded-full bg-primary text-primary-foreground text-xs font-bold flex items-center justify-center shadow-sm hover:bg-primary/80 transition-colors"
    onClick={(e) => {
      e.stopPropagation();
      // Retract the most recent vote on this slot from this voter
      const lastVote = myVotesOnSlot[myVotesOnSlot.length - 1];
      if (lastVote) onRetractVote?.(lastVote.id);
    }}
    title="Click to retract one vote"
  >
    {slotVoteCount}
  </button>
)}
```

Note: The `onRetractVote` callback in the grid receives the **vote ID** (not slotId) because it needs to retract a specific DotVote. The parent (Crazy8sCanvas) will call `retractVote(voteId)` from the store.

IMPORTANT: Do NOT modify slot click behavior when voting is active. The `onSlotClick` still opens EzyDraw. The dedicated "+1" button and badge are separate touch targets with `e.stopPropagation()`.
