**Part A: Create `src/components/canvas/live-cursors.tsx`**

This file exports three items:
1. `CursorBroadcaster` — a renderless component that broadcasts cursor position
2. `LiveCursors` — renders all remote participants' cursors inside ViewportPortal
3. A `CursorArrow` SVG sub-component and `CursorLabel` pill sub-component (internal, not exported)

```typescript
'use client';

import { useCallback, useRef } from 'react';
import { ViewportPortal } from '@xyflow/react';
import { useReactFlow } from '@xyflow/react';
import { useUpdateMyPresence, useOthersMapped } from '@liveblocks/react';
import { shallow } from '@liveblocks/react';
import { Crown } from 'lucide-react';
```

**CursorBroadcaster:**
- Takes no props; returns null (no UI).
- Calls `useUpdateMyPresence()` to get `updateMyPresence`.
- Calls `useReactFlow()` to get `screenToFlowPosition`.
- Provides two callback refs via a forwardRef or callback pattern:
  - `handleMouseMove(e: React.MouseEvent)`: converts `e.clientX, e.clientY` to flow coords via `screenToFlowPosition({ x: e.clientX, y: e.clientY })`, then calls `updateMyPresence({ cursor: flowPos })`.
  - `handleMouseLeave()`: calls `updateMyPresence({ cursor: null })`.
- **Pattern for passing handlers up to parent:** Use a ref-based callback pattern. `CursorBroadcaster` accepts `onHandlersReady: (handlers: { onMouseMove, onMouseLeave }) => void` prop. Call it once in a `useEffect` on mount. Parent stores the handlers in a `useRef` and passes them to `<ReactFlow onMouseMove={...} onMouseLeave={...}>`.

Alternatively (simpler, recommended): `CursorBroadcaster` is NOT a separate component. Instead, export a `useCursorBroadcast()` custom hook that returns `{ handleMouseMove, handleMouseLeave }`. This hook calls `useUpdateMyPresence()` and `useReactFlow()` internally.

**BUT** — since this hook calls Liveblocks hooks, it can only be used inside RoomProvider. In solo mode there is no RoomProvider. So the hook approach requires conditional calling, which violates React rules.

**Recommended approach:** Keep `CursorBroadcaster` as a small component mounted ONLY when `workshopType === 'multiplayer'`. It receives a mutable ref object `handlersRef: React.MutableRefObject<{ onMouseMove: (e: React.MouseEvent) => void; onMouseLeave: () => void } | null>` and populates it. The parent reads from this ref in its `<ReactFlow>` props.

```typescript
interface CursorBroadcasterProps {
  handlersRef: React.MutableRefObject<{
    onMouseMove: (e: React.MouseEvent) => void;
    onMouseLeave: () => void;
  } | null>;
}

export function CursorBroadcaster({ handlersRef }: CursorBroadcasterProps) {
  const updateMyPresence = useUpdateMyPresence();
  const { screenToFlowPosition } = useReactFlow();

  const handleMouseMove = useCallback(
    (e: React.MouseEvent) => {
      const flowPos = screenToFlowPosition({ x: e.clientX, y: e.clientY });
      updateMyPresence({ cursor: flowPos });
    },
    [updateMyPresence, screenToFlowPosition],
  );

  const handleMouseLeave = useCallback(() => {
    updateMyPresence({ cursor: null });
  }, [updateMyPresence]);

  // Populate ref so parent can wire handlers to ReactFlow props
  handlersRef.current = { onMouseMove: handleMouseMove, onMouseLeave: handleMouseLeave };

  return null;
}
```

**LiveCursors:**
- Uses `useOthersMapped` with `shallow` to select `{ cursor, name, color, role }` per connection.
- Renders inside `<ViewportPortal>`.
- Each cursor is a `<div>` with `position: absolute`, `transform: translate(Xpx, Ypx)`, `pointer-events: none`, `transition: transform 80ms linear` (CSS transition for smooth interpolation per research recommendation — zero-dependency, works well at 50ms update rate).
- Contains `<CursorArrow color={color} />` and `<CursorLabel name={name} color={color} isFacilitator={role === 'owner'} />`.

```typescript
export function LiveCursors() {
  const others = useOthersMapped(
    (other) => ({
      cursor: other.presence.cursor,
      name: other.info?.name ?? 'Unknown',
      color: other.info?.color ?? '#6366f1',
      role: other.info?.role ?? 'participant',
    }),
    shallow,
  );

  return (
    <ViewportPortal>
      {others.map(([connectionId, data]) => {
        if (!data.cursor) return null;
        return (
          <div
            key={connectionId}
            style={{
              position: 'absolute',
              transform: `translate(${data.cursor.x}px, ${data.cursor.y}px)`,
              pointerEvents: 'none',
              userSelect: 'none',
              transition: 'transform 80ms linear',
              zIndex: 9999,
            }}
          >
            <CursorArrow color={data.color} />
            <CursorLabel
              name={data.name}
              color={data.color}
              isFacilitator={data.role === 'owner'}
            />
          </div>
        );
      })}
    </ViewportPortal>
  );
}
```
