## TypeScript Compilation Note

When `npx tsc --noEmit scripts/verify-message-id-resolution.ts` is invoked with an explicit file argument, tsc ignores `tsconfig.json` (including `@/*` path alias resolution). This causes 20+ TS2307 errors in `src/db/schema/*.ts` files that import from `@/lib/ids`. This is identical behavior to `scripts/verify-sentinel.ts` — both scripts have the same transitive error profile and zero direct errors. The script runs correctly via `npx tsx` which uses ts-node's module resolver (respects relative imports).
