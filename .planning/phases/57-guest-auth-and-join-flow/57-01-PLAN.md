**1. Create `src/lib/auth/guest-cookie.ts`** — HMAC-SHA256 signed cookie utility for guest identity.

```typescript
import { createHmac, timingSafeEqual } from 'crypto';

const SECRET = process.env.GUEST_COOKIE_SECRET!;
const COOKIE_NAME = 'wp_guest';

export type GuestCookiePayload = {
  participantId: string;  // spar_xxx
  workshopId: string;     // ws_xxx (embedded for cross-workshop validation)
  iat: number;            // issued-at epoch ms
};

export function signGuestCookie(payload: GuestCookiePayload): string {
  const data = JSON.stringify(payload);
  const encoded = Buffer.from(data).toString('base64url');
  const sig = createHmac('sha256', SECRET).update(encoded).digest('base64url');
  return `${encoded}.${sig}`;
}

export function verifyGuestCookie(token: string): GuestCookiePayload | null {
  const dot = token.lastIndexOf('.');
  if (dot === -1) return null;
  const encoded = token.slice(0, dot);
  const sig = token.slice(dot + 1);
  const expectedSig = createHmac('sha256', SECRET).update(encoded).digest('base64url');
  try {
    if (!timingSafeEqual(Buffer.from(sig), Buffer.from(expectedSig))) return null;
  } catch {
    return null;
  }
  try {
    return JSON.parse(Buffer.from(encoded, 'base64url').toString('utf8'));
  } catch {
    return null;
  }
}

export { COOKIE_NAME };
```

Uses Node.js built-in `crypto` (NOT `jsonwebtoken` or `jose` — per research decision). Uses `timingSafeEqual` for timing-attack-safe signature comparison.

**2. Create `src/app/api/guest-join/route.ts`** — POST endpoint that:
- Reads `{ shareToken, displayName }` from request body JSON
- Validates `displayName`: trimmed, 2-30 chars, rejects empty
- Queries `workshopSessions` by `shareToken` — returns 404 if not found
- Queries `workshopSessions` joined to `workshops` to get `workshopId`
- Counts existing `sessionParticipants` for this session to assign color: `PARTICIPANT_COLORS[(count + 1) % PARTICIPANT_COLORS.length]` (index 0 = owner indigo, guests start at 1)
- Generates `liveblocksUserId` via `createPrefixedId('guest')`
- Inserts a `sessionParticipants` record with `clerkUserId: null`, `role: 'participant'`, `status: 'active'`
- Signs the guest cookie with `signGuestCookie({ participantId, workshopId, iat: Date.now() })`
- Sets the cookie via `(await cookies()).set(COOKIE_NAME, signedToken, { httpOnly: true, secure: process.env.NODE_ENV === 'production', sameSite: 'lax', path: '/', maxAge: 60 * 60 * 8 })` — 8-hour lifetime covers a full workshop. Uses `sameSite: 'lax'` (NOT 'strict') per research pitfall #1.
- Returns JSON: `{ ok: true, participantId, workshopId, sessionId (the wses_ id), displayName, color }`

Import `cookies` from `next/headers` (async, await it per Next.js 16.x). Import `PARTICIPANT_COLORS` from `@/lib/liveblocks/config`. Import `sql` from `drizzle-orm` for the count query.
