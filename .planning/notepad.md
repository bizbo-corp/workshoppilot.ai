## Concept Development



Feature on the concept development where I can have a card-level feature to generate all the card's content with an AI 'Generate' button. The feature will have to check if there is existing content already. If there is potentially a dialogue may need to check whether an overwrite is acceptable. 

For each field the user should be offered a button to generate on a field level basis so that rather than completing the whole card, the user can start with the elevator pitch, USP, then each strength, weakness, opportunity, threat, and the feasibility items. 

Furthermore, where a user has entered text, either manually or via generation, there could be a button to 'elaborate' or 'polish' an existing text field.  this option should have an additional text field with instructions that the user can pass to steer direction for the LLM to correct the text. for each section such as elevator pitch, USP, SWOT, and feasibility, there should be a mini system prompt that the LLM can use to generate very relevant answers for those sections.

Consider the ramifications of doing this as a piecemeal process. let me know any considerations I might need to think about and ask any questions before coming up with a full plan.

When a user is generating directly on the whiteboard, we need a way for the LLM to gather context of what's happened via potentially a button in the chat to say "Gather context" or something so that the LLM can pick up where the user has left off.
