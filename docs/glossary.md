## HMAC-signed cookie

An HMAC-signed cookie is a standard web cookie paired with a cryptographic "seal". It verifies that the cookie's data came from your server and hasn't been tampered with by the user. It guarantees data integrity (the data hasn't changed) and authenticity (it was created by the server).

## Race condition

A race condition is a software bug that happens when the behavior of a program depends on the unpredictable timing or sequence of events. It occurs when two or more threads or processes try to access and modify the same "shared resource" (like a variable or file) at the same time. 



## Sliding refresh

A sliding refresh (often called a sliding session or sliding expiration) is a session management and caching technique where a resource's expiration timer automatically resets every time a user interacts with it. As long as you remain active, your session or cached data stays alive.
